# @statusbar/terminal

Output [Statusbar](https://github.com/goto-bus-stop/statusbar) to an ANSI
terminal with [ansi-diff-stream](https://github.com/mafintosh/ansi-diff-stream).

## Usage

```js
const terminal = require('@statusbar/terminal')

bar.use(terminal())
```
